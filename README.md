## Beidou STM32F103 业务代码（可移植）

这份代码先提供“业务层/算法层”，不绑定 Keil 工程和 HAL/StdPeriph，后续你把它嵌入 STM32F103 工程即可。

### 硬件与协议假设（SR2631）

- **引脚**: `PPS`（1Hz 时间标准脉冲输出，可选用）、`RX`、`TX`、`VCC`、`GND`
- **电平**: TTL 串口输出
- **协议**: NMEA-0183
- **默认波特率**: 115200 bps（8N1 常见）

### 目标功能

- MCU 预存一批地标（经纬度、进入阈值、显示文本/语音文本 ID）。
- MCU 持续接收北斗 NMEA 语句，解析出当前位置（优先 RMC / GGA）。
- 计算当前位置与地标距离；距离小于阈值判定“进入地标”，触发事件：
  - 屏幕显示对应信息
  - 播放对应文本（TTS/语音模块由你后续接入）

### 目录结构

- `include/`：对外头文件（可直接放进 Keil 的 include path）
- `src/`：实现文件

### 你嵌入时需要做的最少工作

- 串口中断/DMA 接收字节后，逐字节喂给 `bd_app_on_uart_rx_byte(...)`
- 每隔 50~200ms 调一次 `bd_app_poll(...)`
- 实现回调：
  - `on_enter_landmark(...)`：屏幕显示 + 触发语音播放
  - `on_position_update(...)`（可选）：用于调试显示经纬度/卫星数/定位状态
